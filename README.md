A cross-platform Application for storing User-Data.

    <h3>Prerequisites:</h3>
            - Cmake >= 3.1 <br>

In the root Directory, run these commands:
    
1. ``` mkdir build ```
2. ``` cd build ```
3. ``` cmake .. ```
4. ```make```
5. ``` ./src/app ```

Steps:
    
1. Make The project using Cmake.
